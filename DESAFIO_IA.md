# Avaliação de Uso de Inteligência Artificial

## Contexto

O uso de ferramentas de IA (como GitHub Copilot, ChatGPT, Claude, etc.) é cada vez mais comum no dia a dia de desenvolvimento. Queremos entender como você integraria essas ferramentas ao seu fluxo de trabalho.

## Tarefa

Escolha **uma** das tarefas do desafio técnico e descreva como você usaria uma ferramenta de IA para auxiliar na implementação. Responda as perguntas abaixo em um parágrafo curto para cada uma (não precisa ser extenso).

### Perguntas

1. **Qual tarefa você escolheu e qual ferramenta de IA utilizaria?**
   - Exemplo: "Tarefa 2 (ManifestsController) usando GitHub Copilot no VS Code"

2. **Como você formularia o prompt ou usaria a ferramenta?**
   - Descreva o prompt que daria ou como configuraria a ferramenta
   - Que contexto forneceria (arquivos de referência, documentação, etc.)?

3. **O que você revisaria no código gerado pela IA?**
   - Que tipos de problemas você esperaria encontrar?
   - Como validaria que o código está correto?

4. **Quais os limites do uso de IA neste tipo de tarefa?**
   - Em que cenários a IA ajuda mais? Em quais ajuda menos?
   - Quando você preferiria escrever o código manualmente?
